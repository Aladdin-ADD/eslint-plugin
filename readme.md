# eslint-plugin-autofix

## Install & usage

```bash
$ npm i eslint-plugin-autofix -D
```

add prefix "autofix" to the rulename in eslintrc:
```js
{
  "plugins": ["autofix"],
  "rules": {
    "autofix/no-debugger": "error"
  }
}
```
